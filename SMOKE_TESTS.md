# Smoke Tests

## Test 1: Messaging path
- Send a simple message through Telegram
- Confirm the assistant receives it
- Confirm a response returns through Telegram

## Test 2: Local workflow path
- Trigger a simple file or folder creation workflow
- Confirm the workflow runs locally
- Confirm the result is reported back

## Test 3: End-to-end usefulness
- Trigger a small real task through Telegram
- Confirm the local machine performs the intended action
- Confirm the user receives a useful result message
