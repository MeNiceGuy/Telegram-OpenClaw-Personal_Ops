# Project Structure

- `README.md` — project overview
- `ARCHITECTURE.md` — high-level system design
- `USE_CASES.md` — major workflow categories
- `REAL_EXAMPLES.md` — concrete examples of system use
- `SETUP_NOTES.md` — setup and adaptation notes
- `LIMITATIONS.md` — environment and automation limits
- `PROJECT_GOALS.md` — project intent
- `LICENSE.txt` — license

This repository is intentionally lightweight and focused on documenting a practical AI operations stack rather than pretending to be a universal one-click installer.
