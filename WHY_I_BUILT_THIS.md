# Why I Built This

I wanted something more useful than a normal chatbot.

The goal was to create a personal AI system that could be reached remotely through Telegram and actually support real workflows on a local machine.

Instead of treating AI like a novelty, the focus here is practical use:
- handling real requests
- supporting local workflows
- reducing friction
- making remote control more useful
- building a personal operations stack that feels like a real tool, not just a conversation demo

This project reflects that mindset.
