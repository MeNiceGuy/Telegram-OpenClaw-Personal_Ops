# Limitations

- Personal AI systems are highly dependent on local setup.
- Messaging-based control is powerful, but local execution still depends on environment and tool availability.
- Some workflows are better suited to scripts, some to browser automation, and some to partial manual oversight.
- Not every action can be safely or reliably automated without the right local access and integration layer.
- This project is best presented as a practical blueprint and project story, not as a universal one-click product.
